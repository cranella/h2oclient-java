Provides classes that allow you to interact with an H2O cluster and carry out data modeling tasks. These classes make use of the H2O REST API to access the H2O JVM and its capabilities as a distributed, parallel, in-memory data processing engine. Full H2O REST API spec available here.

Main client classes to use to interface with H2O from within your application:

  • {@link com.att.h2o.H2OConnection H2OConnection}
  • {@link com.att.h2o.H2OFrame H2OFrame}
  • {@link com.att.h2o.H2OModel H2OModel}
  • Subclasses of {@link com.att.h2o.H2OModelBuilder H2OModelBuilder}
